# danalyzer
Modified C code to decode DarkComet


To run the code:
1 - install MinGW. 
2 - compile using:
`gcc .\modified.c -o out.exe -lcomdlg32` # the -lcomdlg32 flag is necessary 
3 - run:
`.\out.exe .\Path\to\your\sample.exe`

A new dialog box will appear printing out all information.
